# matteo-betti-landing

Landing page personale — [matteo-betti.crd.co](https://matteo-betti.crd.co/) redesign

## Deploy

Sito statico in HTML/CSS puro. Nessun build step necessario.

Per preview locale:
```bash
cd matteo-betti-landing && python3 -m http.server 8000
```

## Stack
- HTML5 + CSS custom (no framework)
- Google Fonts: Inter + Playfair Display  
- Zero JS dependencies
- Responsive (mobile-first)
